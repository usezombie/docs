---
title: "Get run"
description: "Fetch run status and scorecard."
api: "GET /v1/runs/{run_id}"
---

## Path parameters

<ParamField path="run_id" type="string" required>
  The unique identifier of the run.
</ParamField>

## Response

<ResponseField name="run_id" type="string">
  Unique identifier for the run.
</ResponseField>

<ResponseField name="status" type="string">
  Current run status. One of `QUEUED`, `RUNNING`, `COMPLETED`, `FAILED`.
</ResponseField>

<ResponseField name="scorecard" type="object">
  Scorecard with dimension-level pass/fail results. Present when the run has completed.
</ResponseField>

<ResponseField name="gate_results" type="array">
  Array of gate evaluation results. Each entry includes the gate name, pass/fail status, and output.
</ResponseField>

<ResponseField name="artifacts" type="array">
  Array of artifact URLs produced by the run (logs, diffs, screenshots).
</ResponseField>
