---
title: "List specs"
description: "List specs in a workspace."
api: "GET /v1/specs"
---

## Query parameters

<ParamField query="workspace_id" type="string" required>
  The workspace to list specs for.
</ParamField>

## Response

Returns an array of spec objects belonging to the workspace.

<ResponseField name="specs" type="array">
  Array of spec objects.
</ResponseField>
