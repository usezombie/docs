UseZombie offers two plans. Both include the full zombie platform — always-on agent hosting, credential firewall, activity stream, webhook ingestion, kill switch. The difference is scale and support.

## Hobby (Free)

Get started with no commitment.

| Feature | Details |
|---------|---------|
| Included credit | $10 |
| Zombies per workspace | Unlimited |
| Event processing | Included |
| Workspaces | 1 |
| Support | Community |

Credit is deducted only for **completed zombie executions** — the per-event agent reasoning that produced a response. If an event fails before the agent produces output (budget breach, skill error, webhook rejected), it is free.

## Scale (Pay as you go)

Everything in Hobby, plus:

| Feature | Details |
|---------|---------|
| Workspaces | Unlimited |
| Priority event queue | Included |
| Cost control dashboards | Included |
| Support | Email |

Scale billing is usage-based. You are charged for compute and token consumption on completed zombie executions. The same rule applies: failed executions are free.

## Choosing a plan

Start with Hobby. When you need more than one workspace, priority queue access, or email support, upgrade to Scale from the dashboard or CLI:

```bash
zombiectl workspace upgrade-scale --workspace-id ws_abc --subscription-id sub_xxx
```
